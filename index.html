que un sofware ?

el software es la parte intangible de un computador tambien

estan relacionadas con las aplicaciones o programas que contienen código que permiten que

una computadora o dispositivo electrónico realice tareas específicas dentro

de un computador

Software de Sistema

es un conjunto de programas diseñados para la ejecución de otros programas y de garantizar la estabilidad del sistema tambien gestiona y controla los recursos y actividades del hardware de la computadora.

Software de Aplicación

es un conjunto de categoria de programas informaticos que ayudan a los usuarios a realizar tareas especificas y determinadas como por ejemplos Microsoft Word, Google Chrome, Adobe

Software de Desarrollo (o de Programación)

son conjunto de herramientas que le permiten a los programadores a Desarrollollar programas de informaticas usando diferentes leguajes segun sea su necesidad de una manara practica y facil utiles para realizar diversas actividades.

algun mecanimos fisico que permite que comparta informacion y recursos

internet son totalmente heterogeneas;

Estos servicios permiten la interacción,comunicacion, almacenamiento y

Líneas telefónicas

Se puede utilizar una línea telefónica que se subdivide en líneas telefónicas

convencionales (transmite señales de forma analógica pero

actualmente fueron desplazadas por otras que ofrecen mayores beneficios)

o digitales (mediante el empleo de un adaptador de red se traducen las tramas

resultantes de la computadora a señales digitales).

Cable (fibra óptica)

se refiere a un tipo de cable utilizado para transmitir datos

a través de fibras ópticas, que son filamentos de vidrio o plástico

extremadamente delgados que transmiten información en forma de pulsos de luz.

Satelital

a través del internet satelital se envia una señal desde el satélite hasta una

antena parabólica instaladas en la empresa o en el hogar o negocio de nuestros clientes quienes desean obtenerla

que a su vez se conecta a un módem Wi-Fi.

Lo anterior permite la conexión de cualquier dispositivo de forma inalámbrica.

Redes inalámbricas

Estas redes utilizan tecnologias avanzadas de transmision

de señales de radio o microondas para establecer conexiones

y facilitar la comunicación entre dispositivos, como computadoras,

telefonos celulares impresoras etc atravez de ondas electromagnéticas.

Lineas electricas

Las lineas eléctricas, en si mismas no están directamente relacionadas

con el funcionamiento del internet sin embargo

hay aspectos indirectos de la infraestructura eléctrica que pueden afectar

la disponibilidad y el funcionamiento de los servicios de internet

Aquí hay algunas maneras en que las líneas eléctricas pueden estar relacionadas con el internet:
